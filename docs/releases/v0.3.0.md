# ArchMap v0.3.0

ArchMap v0.3.0 adds the containment-first **Topology** view and sharpens the
authoring distinction between exclusive zones and structural subgraphs.

## Highlights

- New `topology` base view with a golden-ratio canvas and deterministic integer-cell packing.
- Components can span merged grid cells while remaining centered and overlay-independent.
- Topology zones use translucent palette fills, never intersect as siblings, and retain visible clearance.
- Nested parent-child zones remain supported as intentional containment.
- Subgraphs render as unfilled dashed structural guides and may intersect.
- Component-safe orthogonal routing and rendered endpoint validation apply to Topology.
- New multi-region Topology demo and updated AI authoring guidance.
- Android platform stack sample now uses `graph LR` with the Layer view.

## Authoring Rule

Use a `zone` when overlap would make placement or ownership ambiguous. Use a
`subgraph` for non-exclusive structural grouping such as a flow, feature slice,
or source hierarchy.
